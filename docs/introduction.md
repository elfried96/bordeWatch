# Introduction à BorderWatch

**BorderWatch** est un projet de surveillance intelligente pour les zones frontalières à l'aide de technologies de vision par ordinateur. Le projet utilise le framework Django pour le backend, et des modèles de détection basés sur YOLOv8 pour identifier des objets ou des événements en temps réel à partir de flux vidéo ou d'images.

### Objectifs principaux :
- Détection automatique d'intrusions et d'objets dans les zones frontalières.
- Fourniture d'une interface web pour la visualisation des alertes.
- Surveillance en temps réel à partir de caméras de drones ou de caméras de sécurité.
