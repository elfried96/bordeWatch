# BorderWatch

**BorderWatch** est un projet Django conçu pour la surveillance des frontières avec des technologies de vision par ordinateur.

## Prérequis

- **Python 3.8+**
- **Django** (installé via `pip`)
- Un compte **GitHub**
- **Ubuntu** (OS utilisé pour le développement)
